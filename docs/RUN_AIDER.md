# Aider chat && code
Ce projet utilise aider pour l'aide au développement et l'assistance dans certains domaines de code et de documentation.

# Prérequis
* Activer le virtualenv du projet via les commandes `pyhton -m venv .venv` et `source .venv/bin/activate`.
* Utiliser le `pip install -r requirements.txt` du projet

# Lancement
Utiliser les commandes suivantes :
```bash
aider
/read-only docs/INSTRUCTIONS.md
```
